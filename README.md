# 🚀 Portfolio de Ricardo Sandjon

Bienvenue sur mon portfolio ! Je suis **Développeur Web Fullstack Junior** passionné par la création d'interfaces modernes, performantes et accessibles.

## 🛠️ Stack Technique

- **Frontend :** React 18, Vite, Sass (SCSS), Framer Motion
- **Backend :** Node.js, Express
- **Base de données :** MongoDB
- **Outils :** EmailJS, GitHub Pages, ESLint

## 🌟 Fonctionnalités

- **Responsive Design :** Optimisé pour Mobile, Tablette et Desktop.
- **Animations fluides :** Utilisation de Framer Motion pour une expérience utilisateur premium.
- **Formulaire de contact :** Intégration sécurisée avec EmailJS (Variables d'environnement).
- **SEO & Accessibilité :** Score Lighthouse optimisé.
